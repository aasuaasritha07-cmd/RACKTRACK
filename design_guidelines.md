# Design Guidelines for RackTrack-Inspired Home Page

## Design Approach
**Reference-Based Approach**: Recreating the exact visual design of RackTrack's hero section with dark theme, gradient backgrounds, and AI-focused messaging.

## Core Design Elements

### A. Color Palette
**Dark Mode Primary:**
- Background: Deep dark blue/black gradient (220 25% 8% to 220 20% 4%)
- Primary blue: Bright cyan-blue (200 90% 60%)
- Text primary: Pure white (0 0% 100%)
- Text secondary: Light gray (220 15% 70%)
- Accent gradient: Blue to cyan (200 90% 60% to 180 85% 55%)

### B. Typography
- **Primary Font**: Modern sans-serif (Inter or similar via Google Fonts)
- **Hero Headline**: Bold, large scale (text-4xl to text-6xl)
- **Body Text**: Medium weight, readable sizing (text-lg)
- **Navigation**: Medium weight, clean styling

### C. Layout System
**Spacing System**: Tailwind units of 4, 8, 12, 16, 24 for consistent rhythm
- Container max-width with centered alignment
- Generous vertical spacing between sections
- Balanced horizontal padding

### D. Component Library

**Navigation Bar:**
- Dark background with subtle transparency
- Logo positioned left, menu items right
- Clean, minimal styling with hover effects

**Hero Section:**
- Large centered headline with gradient text treatment
- Descriptive subtext below headline
- Four corner icons positioned around central element
- Central circular icon with blue gradient background
- Ample whitespace for breathing room

**Company Logos Section:**
- Horizontal layout with even spacing
- Subtle gradient background overlay
- Logos in muted colors for sophisticated appearance

**Icons:**
- Line-style icons for corner elements (Rack, Cable, Port, Switch)
- Central arrow/direction icon with gradient treatment
- Consistent sizing and visual weight

### E. Key Visual Elements

**Gradients:**
- Background: Subtle dark blue gradient from top to bottom
- Central icon: Bright blue to cyan radial gradient
- Text highlights: Blue gradient for emphasis

**Layout Structure:**
- Full-width hero section
- Centered content with max-width container
- Four-quadrant icon arrangement around central focal point
- Bottom section with company logos on gradient background

## Images
**Hero Background**: Dark gradient background (no large hero image needed)
**Central Icon**: Blue gradient circular background with arrow/directional icon
**Corner Icons**: Four minimalist line icons representing infrastructure components
**Company Logos**: Footer section with partner/client logos (AWS, ServiceNow, Equinix, Cisco, Smart)

The design emphasizes clean, modern aesthetics with strategic use of blue gradients against dark backgrounds, creating a professional tech-focused appearance that conveys AI innovation and infrastructure management expertise.
